Manuel d'utilisation:

Premièrement le serveur est multi-client, vous pouvez donc lancer plusieurs fois la classe client afin de lancer plusieurs clients.
Le serveur va gérer les clients en leurs attribuant un identifiant unique et en récupérant le nom du client qui se connecte.
Pensez bien à mettre votre bonne addresse et port dans les paramètres des sockets. Par défaut les paramètres sont: "127.0.0.1" et 5000.

Comment on utilise ce serveur ?

-Lancez le serveur et ensuite le/les client(s)
-Entrez le/les nom(s) des clients.
-Vous pouvez discuter entre vous.
-Tapez la commande "quit" dans un client si vous voulez fermer manuellement la connexion avec le serveur et ce client.
-Si vous tapez deux fois le même message, le serveur fermera automatiquement la connexion avec ce dernier.
-Tant que le serveur est lancé vous pouvez vous reconnecter autant de fois que vous le voulez.
